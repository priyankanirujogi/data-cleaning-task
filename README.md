# Data Cleaning & Preprocessing Task

## Objective
To clean and preprocess raw data for machine learning.

## Dataset
Titanic dataset (loaded using seaborn)

## Steps Performed
- Checked missing values
- Filled missing values using mean and mode
- Converted categorical data into numerical
- Applied normalization
- Visualized outliers using boxplot

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

## Output
Cleaned dataset saved as cleaned_titanic.csv
